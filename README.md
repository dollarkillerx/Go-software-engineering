## 单机
### 简单的业务场景
- 红包业务
    - 红包:一定数量和金额的红包序列
        - 红包是具备虚拟资金特征的特殊商品
        - 发红包和收红包实际上是资金交易过程
        - 资金交易是资金从一个账户流向另一个账户的过程
